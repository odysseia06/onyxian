# Lattice Crypto On-Ramp

Ordered run for getting genuinely fluent in SIS/LWE-land this winter, paced against [[Academic/Courses/EECS-598 Lattice-Based Cryptography/00 Overview|EECS-598]]. The order is deliberate: map first, then sources.

## The List

1. [[Peikert2016 - A Decade of Lattice Cryptography]] — read Chapters 1–4 before anything else; it fixes the notation the rest of the list is read in. *(summarized 2025-12-31)*
2. [[Regev2005 - Learning with Errors]] — the source for the problem and the quantum reduction; short enough to read fully once the survey has set the stage. *(summarized 2025-12-23)*
3. [[Gentry2008 - Trapdoors for Hard Lattices]] — the trapdoor toolbox; pairs with course weeks 6–8. *(reading now)*
4. Micciancio–Peikert 2012, "Trapdoors for Lattices: Simpler, Tighter, Faster, Smaller" — not in the library yet; intake when the GPV pass is done, because it is best read as a critique of GPV's trapdoor.
5. Lyubashevsky–Peikert–Regev 2010, "On Ideal Lattices and Learning with Errors over Rings" — queue for the ring weeks; the survey's Chapter 4 fine print is the reading guide.

## Why This List Exists

- The course covers the same arc but waves at the proofs; this run is where the proofs actually get read.
- Cross-thread: everything here feeds the [[Learning with Errors]] topic note.
