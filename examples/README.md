# Example vaults

Reference vaults, one per profile, **generated by the engine and never hand-edited** (KICKSTART.md D6, §11). Regenerate with `python tools/gen_examples.py` (`ONYX_NOW` pinned, so trees are byte-identical on every machine and OS); CI reruns the script and fails on any drift, which makes every example a standing integration test.

| Example | Profile | Modules |
|---|---|---|
| `minimal/` | `minimal` | core |
| `fitness-focused/` | `fitness-focused` | core, daily-notes, fitness |
| `student/` | `student` | core, daily-notes, academic |

Browse one in Obsidian to see exactly what a fresh `onyx init` produces — including the `.claude/` agent surface, which the vault works fully without (P2).
